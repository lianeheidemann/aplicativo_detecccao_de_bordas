# 📷 Detector de Bordas

Aplicativo mobile desenvolvido em Flutter para captura de imagens utilizando a câmera do celular e aplicação de técnicas básicas de processamento de imagens, incluindo conversão para tons de cinza e detecção simples de bordas.

Projeto desenvolvido para a disciplina **Laboratório de Programação Mobile** do curso de **Ciência da Computação**.

---

## Informações Gerais

O objetivo do aplicativo é permitir que o usuário capture uma foto utilizando a câmera do dispositivo e visualize automaticamente:

- Imagem original capturada;
- Imagem convertida para escala de cinza;
- Imagem processada com detecção simples de bordas.

A proposta busca introduzir conceitos básicos de:

- Processamento de imagens;
- Visão computacional;
- Manipulação de pixels;
- Desenvolvimento mobile com Flutter.

---

## Tecnologias Utilizadas

Projeto desenvolvido com:

- **Flutter**
- **Dart**
  
---

## Funcionalidades

✔ Captura de imagem pela câmera do celular  

✔ Conversão automática para tons de cinza  

✔ Aplicação de algoritmo simples de detecção de bordas  

✔ Exibição da imagem original e processada  

✔ Interface simples para navegação entre captura e resultado
